I set out to learn about the functional programming paradigm by recreating a familiar game.
I hoped to incorporate some of the principles laid out by Scott Wlaschin on his website F# for fun and profit. My goal was to use domain driven design to create an algebraic type system that was simple yet flexible. I also wanted to use Railway Oriented Programming to handle move validation.
